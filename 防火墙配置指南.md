1. **右键点击** `setup_lan_access.bat`
2. **选择** "以管理员身份运行"
3. **按提示操作** 完成配置

## 方法二：手动配置

### 1. 打开Windows防火墙高级设置
- 按 `Win + R`，输入 `wf.msc`，回车
- 或者：控制面板 → 系统和安全 → Windows Defender 防火墙 → 高级设置

### 2. 添加入站规则
#### 配置端口5000（后端）：
1. 点击左侧"入站规则"
2. 点击右侧"新建规则"
3. 选择"端口" → 下一步
4. 选择"TCP"，输入"5000" → 下一步
5. 选择"允许连接" → 下一步
6. 勾选所有配置文件 → 下一步
7. 名称输入"会议室预约系统-后端" → 完成

#### 配置端口8080（前端）：
1. 重复上述步骤
2. 端口改为"8080"
3. 名称改为"会议室预约系统-前端"

### 3. 验证配置
运行 `test_network.bat` 检查配置是否成功

## 方法三：命令行配置（需要管理员权限）

```cmd
# 以管理员身份运行CMD，然后执行：
netsh advfirewall firewall add rule name="会议室预约系统-后端" dir=in action=allow protocol=TCP localport=5000
netsh advfirewall firewall add rule name="会议室预约系统-前端" dir=in action=allow protocol=TCP localport=8080
```

## 测试局域网访问

### 在同一WiFi网络的其他设备上测试：

1. **手机测试**：
   - 连接同一WiFi
   - 浏览器访问：http://172.100.12.17:8080
   - 或扫描二维码：meeting_room_qr.png

2. **电脑测试**：
   - 连接同一WiFi
   - 浏览器访问：http://172.100.12.17:8080

### 如果无法访问，检查：

1. **防火墙设置**：确保端口5000和8080已开放
2. **网络连接**：确保设备在同一WiFi网络
3. **服务状态**：确保系统已启动（运行start_system.bat）
4. **路由器设置**：确保路由器未启用设备隔离
5. **杀毒软件**：检查是否被杀毒软件阻止

## 常见问题解决

### 问题1：无法访问网站
**解决方案**：
1. 检查防火墙规则是否正确添加
2. 重启系统服务
3. 检查IP地址是否正确

### 问题2：页面加载但功能异常
**解决方案**：
1. 检查后端服务是否正常运行
2. 查看浏览器控制台是否有错误
3. 确认API地址配置正确

### 问题3：手机无法访问
**解决方案**：
1. 确认手机连接的是同一WiFi网络
2. 尝试关闭手机的移动数据
3. 清除浏览器缓存后重试

## 验证配置成功

当配置成功后，你应该能够：
1. 在本机访问：http://localhost:8080
2. 在局域网其他设备访问：http://172.100.12.17:8080
3. 手机扫描二维码正常访问
4. 所有功能正常使用（预约、查询、管理）
